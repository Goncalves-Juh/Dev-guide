💻 Terminal/CLI

Comandos essenciais para linha de comando

ls listar arquivos e pastas
pwd mostrar diretório atual
echo Hello World imprimir texto na tela
clear limpar a tela do terminal
date mostrar data e hora atual
whoami mostrar usuário atual
help mostrar comandos disponíveis
js console.log('Hello') executar código JavaScript
calc 2 + 2 calculadora matemática
cd projeto navegar para diretório

🏗️ HTML

Estrutura de páginas web

<!DOCTYPE html> declaração do tipo de documento
<html> elemento raiz da página
<head> metadados da página
<title> título da página
<body> conteúdo visível da página
<h1> cabeçalho principal
<p> parágrafo de texto
<div> container genérico
<span> elemento inline genérico
<a href=""> link para outra página

🎨 CSS

Estilização de páginas web

color cor do texto color: blue;
background-color cor de fundo background-color: #f0f0f0;
font-size tamanho da fonte font-size: 16px;
padding espaçamento interno padding: 10px;
margin espaçamento externo margin: 20px;
display tipo de exibição display: flex;
width largura do elemento width: 100%;
height altura do elemento height: 50px;
